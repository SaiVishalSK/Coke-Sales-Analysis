This project presents a complete sales and profitability analysis of Coca-Cola retail performance using Microsoft Excel.
The goal is to understand how Coca-Cola performs across regions, retailers, time periods, and pricing, and to generate insights for business decision-making.

All analysis, calculations, and dashboards were built entirely in Excel.
